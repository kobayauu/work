autoGUI
◎機能
操作開始・・・シナリオファイルに従い、PCの自動操作(マウス,キーボード操作)を行う
取得開始・・・マウスでクリックした位置の座標を取得及び表示する

●ファイル定義
操作ファイル・・・操作の内容が記述するシナリオファイル(テキスト形式)
　　　　　　　　　1行目から順に指定されたPC操作(マウス,キーボード操作)を行う
　　　　　　　　　内容の詳細は「▲シナリオ定義」参照

関数ファイル・・・操作ファイルで繰り返し実施する内容を記述するシナリオファイル(テキスト形式)
　　　　　　　　　操作ファイルで使用する【func】コマンドの各関数の内容をここに記述する
　　　　　　　　　内容の詳細や各コマンドは操作ファイルと同様だが、関数同士の区切りは【空白行】とする
　　　　　　　　　※関数ファイルを使用しなくてもPC操作は可能


▲シナリオ定義
カンマ(,)区切りで記述する。各列での意味は左から順に下記となる
【コメントアウト,コマンド(関数名),引数1,引数2,引数3,引数4,メモ】
コメントアウト　・・・任意の文字を記述すると、この行で指定した操作をスキップする
コマンド(関数名)・・・実施する操作を指定する。詳細は「■コマンド定義」参照
　　　　　　　　　　　関数ファイルでは任意の関数名を指定する
引数1～4      　・・・コマンドで必要な各種情報を指定
メモ          　・・・操作内容のメモ欄。操作に影響しない


■コマンド定義
Sleep   ・・・指定した時間待機する
　　   　　　 引数1：待機時間(ms)

Message ・・・メッセージボックスを表示する
　　　　　　　ボックスで「はい」ボタンをクリックした場合、操作を中断する
　　   　　　 引数1：表示文字

Click   ・・・指定した座標でマウスクリックを行う
　　　　　 　 引数1：X座標, 引数2：Y座標, 引数3：クリック回数

Input　 ・・・指定した文字の入力を行う
　　　　　　　引数1：入力文字

KeyPress・・・指定したキーボードのキー押下を行う(2つ同時まで可)
　　　　　 　 引数1：キー1, 引数2：キー2(未指定でも可)

Drug    ・・・指定した始点座標から終点座標へドラッグを行う
　　　　　　　引数1：始点X座標, 引数2：始点Y座標, 引数3：終点X座標, 引数4：終点Y座標

Copy    ・・・「Ctrl+c」を用いてコピーし、コピーした文字列を出力結果に表示する
　　　　　　  事前にコピーしたい文字を選択しておく必要がある
　　　　  　  引数1：メモ用の追加表示文字(未指定でも可)

OpenFile・・・指定したファイルを開く
　　　　　　　引数1：ファイルパス(フルパス)

Func    ・・・指定した関数の動作を行う
　　　　　　　関数の詳細内容は別ファイルの記述する
　　　　　　　引数1：関数名

【例】
・操作ファイル
,Func,ファイルを開く,,,,
,Click,100,100,1,,ファイル入力欄選択
,Input,test.txt,,,,ファイル名入力
,Click,200,200,1,,ボタン1クリック
,Click,300,300,2,,ファイルパス選択
,Copy,ファイルパス,,,,ファイルパスをコピー
,Click,400,400,1,,保存ボタンクリック←ここの行のコマンドは実行されない
,Func,ファイルを閉じる,,,,
・・・

・関数ファイル
,ファイルを開く,,,,,←上記の操作ファイルで使用する関数名を記述
,Click,100,100,1,,
,OpenFile,test.exe,,,,textアプリ起動
,,,,,,←関数同士の区切りは空白行にとする
,ファイルを閉じる,,,,,←上記の操作ファイルで使用する関数名を記述
,Click,200,200,1,,閉じるボタンクリック
,Click,300,300,1,,確認ダイアログの「OK」ボタンクリック
・・・
